Установка и запуск:

Убедитесь, что у вас установлен Python (рекомендуется версия 3.7+).
Откройте терминал (командную строку) и перейдите в папку с файлами приложения.
Установите зависимости, введя команду:

pip install -r requirements.txt


1. Запустите приложение, введя команду:

python main.py

Или запустите

start.bat

Использование:

После запуска приложения вас встретит меню с возможными операциями:

Добавить книгу
Поиск книги
Менеджер Удаления
Редактировать книгу
Вывести список всех книг
Выйти из приложения
